### <span style="color:#A855F7">⭐ Bilder in CodeSandbox hochladen Option 1</span>

1. Rechtsklick auf den Ordner öffentlich und Upload auswählen.
2. Die Bilddatei vom PC auswählen zum Beispiel Maler Login Shield.png.
3. Prüfen ob die Datei korrekt angezeigt wird indem das Icon normal aussieht kein Fehlertext erscheint und die Datei sichtbar im Ordner liegt.
4. In React das Bild über backgroundImage mit url und dem Dateinamen referenzieren.
5. Optional testen indem in der Vorschau rechts die URL mit dem Dateinamen eingegeben wird und das Bild angezeigt wird wenn alles korrekt ist.

---


### <span style="color:#A855F7">⭐ Bilder in CodeSandbox hochladen Option 2</span>

1. Das Bild direkt per Drag and Drop in den Ordner öffentlich ziehen damit es automatisch hochgeladen wird.
2. Warten bis die Datei vollständig erscheint und das normale Icon angezeigt wird.
3. Prüfen ob kein Fehlertext sichtbar ist und die Datei korrekt im Ordner liegt.
4. Das Bild in React über backgroundImage mit url und dem Dateinamen referenzieren.
5. Optional testen indem in der Vorschau rechts die URL mit dem Dateinamen eingegeben wird und das Bild angezeigt wird wenn alles korrekt ist.
