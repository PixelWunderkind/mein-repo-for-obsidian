# ✅ **Die 4 Prompts — in der korrekten Reihenfolge**

## **1) Prompt 1 — Bild-Analyse-Tool Kimi K2.6**

**Zweck:** Bild analysieren → Stil extrahieren → Grundlage für Kimi schaffen.  
**Phase:** Vor der Code‑Generierung.

---

## **2) Prompt 2 — Pinterest Reference Framework**

**Zweck:** Prüfen, ob das Bild überhaupt geeignet ist.  
**Phase:** Direkt vor Prompt 1.  
**Funktion:** Qualitätsfilter für Referenzbilder.

➡️ **Ja, dieser ist der zweite Prompt.**

---

## **3) Prompt 3 — Technischer Reparatur‑ und Veredelungs‑Prompt (Claude 4.7)**

**Zweck:** Kimi‑Code reparieren, Parallax einbauen, Motion fixen, Cursor, Lenis.  
**Phase:** Nach Kimi, nach `npm run dev`.

---

## **4) Prompt 4 — SaaS‑Upgrade‑Prompt**

**Zweck:** Backend hinzufügen (Supabase, Stripe, Emails, Dashboard).  
**Phase:** Optional, nach Prompt 3.

---

# 🎯 **Kurz: Deine Pipeline in der richtigen Reihenfolge**

1. **Prompt 1** → Bild analysieren
2. **Prompt 2** → Bildqualität prüfen
3. **Prompt 3** → Code reparieren & veredeln
4. **Prompt 4** → Backend hinzufügen (optional)
