## 🛠 Workflow: MP4 zu Animiertem WebP (Parallax) via EzGif

### 1. Upload & Initialisierung

* **URL:** [ezgif.com/video-to-webp](https://ezgif.com/video-to-webp)
* **Aktion:** Datei auswählen und auf **Upload Video!** klicken.

### 2. Konvertierungseinstellungen

Stelle folgende Parameter für eine optimale Balance zwischen Qualität und Performance ein:

| Parameter | Wert |
| --- | --- |
| **Frame rate (FPS)** | 24 |
| **Size** | Up to 1280px |
| **Method** | FFmpeg |
| **Quality** | 85 |
| **Loop count** | 0 (unendlicher Loop) |

* **Aktion:** Klicke auf den blauen Button **Convert to WebP!**.

### 3. Frame-Extraktion (Optional/Split)

Falls die Einzelbilder benötigt werden:

1. Wähle unter dem generierten Video die Option **Split**.

2. Setze das **Output Format** auf **WebP**.
 
3. Klicke auf **Split to Frames!**.
 
4. Klicke auf den roten Button **Download Frames as ZIP**.
