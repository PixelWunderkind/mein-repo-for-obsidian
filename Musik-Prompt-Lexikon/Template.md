
Um die Rückverlinkung sauber und funktional in deine Notiz **„Der KI-Songwriter-Shaping-Prompt“** einzufügen, nutze einfach die interne Wiki-Link-Syntax von Obsidian.

Kopiere diesen Block und füge ihn am Ende deiner Notiz ein:

Markdown

```
---
**Zurück zur Übersicht:** [[Übersicht-meiner-verschiedenen_Musik_Styles]]
```

### Profi-Tipp für Obsidian:

Falls du möchtest, dass der Link schöner aussieht (z. B. ohne die Unterstriche), kannst du einen **Alias** verwenden:

Markdown

```
**Zurück zur Übersicht:** [[Übersicht-meiner-verschiedenen_Musik_Styles|Musik-Styles Übersicht]]
```

Sobald du den Link eingefügt hast, kannst du mit `Strg + Klick` (oder einfach Klick im Lesemodus) sofort zwischen deinem neuen Shaping-Prompt und deiner Style-Sammlung hin- und herspringen.


### Version 3: Komplett ohne Unterstriche (Alias-Variante)

Kopiere diesen Text, wenn der Link im Lesemodus wie ein normaler Satz ohne Unterstriche aussehen soll:

Markdown

```
**Zurück zur Übersicht:** [[Übersicht-meiner-verschiedenen_Musik_Styles|Übersicht meiner verschiedenen Musik Styles]]
```

**Erklärung:**

Obsidian leitet dich intern zur Datei mit den Unterstrichen weiter, aber deine Augen sehen in der Notiz nur den Text: **Übersicht meiner verschiedenen Musik Styles**.


**Zurück zur Übersicht:** [[Übersicht-meiner-verschiedenen_Musik_Styles|Übersicht meiner verschiedenen Musik Styles]]
